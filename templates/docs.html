Documentación de la API

URL Base

http://127.0.0.1:5000

Endpoints

Inicializar Base de Datos

GET /db/init

Inicializa la base de datos creando todas las tablas.

Eliminar Tablas de la Base de Datos

GET /db/drop

Elimina todas las tablas de la base de datos.

Agregar un Personaje

POST /character/add
POST /character/new
POST /character

Agrega un nuevo personaje a la base de datos.

Cuerpo de la Solicitud:

{
    "character_id": 1,
    "name": "Luke Skywalker",
    "height": 172.0,
    "mass": 77.0,
    "hair_color": "blond",
    "skin_color": "fair",
    "eye_color": "blue",
    "birth_year": 1998
}

Obtener un Personaje por ID

GET /character/get/<int:character_id>
GET /character/<int:character_id>

Recupera un personaje de la base de datos por su ID.

Obtener Todos los Personajes

GET /character/getAll
GET /character

Recupera todos los personajes de la base de datos.

Eliminar un Personaje por ID

DELETE /character/delete/<int:character_id>

Elimina un personaje de la base de datos por su ID.
